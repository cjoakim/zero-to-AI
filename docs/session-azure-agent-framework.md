## AI Orchestrators 

- [LangChain](https://www.langchain.com)
- [LangGraph](https://www.langchain.com/langgraph)
- [Semantic Kernel](https://learn.microsoft.com/en-us/semantic-kernel/overview/)  (Microsoft)
  - Released March 2023
  - Created by Microsoft Research (MSR)
  - Support for both .NET and Python
- [AutoGen](https://www.microsoft.com/en-us/research/project/autogen/)  (Microsoft)
  - Released October 2023
  - Also created by Microsoft Research (MSR)
- [Agent Framework](https://github.com/microsoft/agent-framework)  (Microsoft)
  - [GitHub](https://github.com/microsoft/agent-framework)
  - Microsoft's comprehensive multi-language framework for building, orchestrating, and deploying AI agents
  - It merges and enhances Semantic Kernel and AutoGen
  - Support for both .NET and Python implementation
  - Public preview on October 1, 2025
  - Reached general availability (GA) on October 2, 2025 !? 
  - The Microsoft Agent Framework is primarily maintained and developed by the Azure AI Foundry team

## Microsoft Agent Framework

Agent Framework offers two primary categories of capabilities:

- **Agents**: Self-contained AI agents that can perform tasks independently or in collaboration with other agents
- **Workflows**: Orchestrate multiple agents to perform complex tasks
